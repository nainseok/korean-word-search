# korean-crossword-helper

## 개요
한글을 이용한 십자말풀이 또는 그냥 미궁 문제를 만들 때,
특정 조건을 만족하는 단어를 편하게 찾을 수 있도록 프로그램을 모아 두었습니다.  

## 파일 설명
### words 폴더
단어들이 정리된 텍스트 파일을 모아둔 폴더입니다.  
standard/easy : 각각 표준국어대사전/한국어기초사전입니다.  
immutables : 불변어만 모아둔 파일입니다.  

### word_search
와일드카드를 이용해 단어를 검색할 수 있습니다.  
지금은 ?만 이용 가능하지만 *도 추후 추가 예정입니다.  

### only_using_these
입력한 글자만 사용해서 만들 수 있는 단어를 검색할 수 있습니다.  

### decompose/find_scramble
decompose 파일은 글자를 기본 자모로 분리합니다.  
find_scramble은 입력한 단어를 기본 낱자 단위로 분리한 뒤 재조합해서 만들 수 있는 단어를 알려줍니다.
